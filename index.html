Curso de Inclusión en el Transporte Público

Capacitación clara, profesional y accesible para conductores STP.

Un portal educativo para aprender buenas prácticas, normativa, empatía y toma de decisiones en situaciones reales del transporte público, con acceso disponible en todo momento como parte de la propuesta del proyecto.

Objetivo

Un transporte público más inclusivo comienza con conductores mejor preparados.

La experiencia guía al usuario desde el login hasta la descarga del certificado, con módulos cortos, simulación 360, evaluación final y disponibilidad 24/7 para avanzar sin depender de horarios.

Alcance académico

Propuesta pensada para funcionar como plataforma siempre disponible.

Para la presentación universitaria, el acceso 24/7 se plantea como una característica del sistema: el conductor puede entrar cuando lo necesite, retomar actividades y revisar su avance desde cualquier dispositivo.

24/7

Acceso permanente

El curso estaría disponible fuera del horario laboral para facilitar la capacitación.

Flexible

Avance por etapas

Los módulos permiten pausar, continuar y completar el proceso de forma ordenada.

Certificado

Cierre verificable

Al finalizar, el usuario obtiene un certificado como evidencia del aprendizaje.

Acceso público seguro

La web puede ser visible para cualquier persona sin exponer datos ni servidores propios.

Para este prototipo universitario, la opción más segura es publicarlo como sitio estático. Cualquier persona puede verlo, pero la página no guarda RUT, contraseñas, correos ni respuestas reales.

Sin base de datos No existe información sensible almacenada.
Sin backend propio No se expone un servidor del equipo a internet.
Modo invitado El acceso funciona como demostración abierta del flujo.
Solo contenido público Ideal para entregar y presentar el proyecto académico.
