# Portal de Capacitacion STP

Prototipo web academico y funcional para cursos de inclusion en el transporte publico.

## Como ver la pagina

Abrir `index.html` en el navegador.

## Publicacion recomendada

Este proyecto esta preparado para GitHub Pages como sitio estatico.

1. Crear un repositorio en GitHub.
2. Subir estos archivos a la rama `main`.
3. En GitHub, ir a `Settings > Pages`.
4. Seleccionar `Deploy from a branch`.
5. Elegir rama `main` y carpeta `/root`.
6. Guardar.

GitHub generara un link publico parecido a:

`https://usuario.github.io/nombre-del-repo/`
